# Pipax

## Introduction
Experiments on pipeline parallelism using Jax

## Dependency

The environment for testing the project:
```
python >= 3.8.10
jax==0.2.25
jaxlib==0.1.74+cuda11.cudnn805
optax==0.1.0
pytest==6.2.4
tensorboard==2.6.0
```
You may want to change the CUDA/cudnn version to match your device.
